配置github账号的SSH key

生成ssh公钥

三次回车即可生成ssh key，对于存放目录，可以使用默认，也可在默认的基础上添加自定义目录。个人倾向于自定义目录。

查看public key，并把它添加到github公钥中

将#号后的内容拷贝到 Github 公钥中并保存即可。

具体可参考Github官网的配置，或参考码云关于ssh key的配置
